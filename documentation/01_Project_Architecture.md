# AI-Powered CAT Modelling System

## Project Overview

The AI-Powered CAT Modelling System is an enterprise-level insurance and reinsurance application designed to assess catastrophe risks, manage insurance policies, process claims, and support data-driven decision making using Artificial Intelligence and Machine Learning.

---

## Project Modules

1. User Management
2. Property Management
3. Insurance Policy Management
4. Catastrophe Event Management
5. Risk Assessment
6. Claims Management
7. Reinsurance Management
8. AI Prediction Engine
9. Reporting Module
10. Dashboard & Analytics
